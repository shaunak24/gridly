# v2.1

## Goal

Polish the multi-game platform with Grid Snap post-game UX improvements, game icons, and difficulty settings.

## Status

**Complete**

## Features

### Grid Snap polish

- [x] On win, the puzzle grid **stays in the same in-viewport position** as during play; the end bar and modal appear below without jumping the image off-screen.
- [x] Tile dragging is **disabled** after the puzzle is solved.
- [x] The completed puzzle shows the **full image without grid lines or tile borders**.
- [x] **Play again** after win remounts the puzzle canvas cleanly (no gesture/Reanimated crash).
- [x] **Game icons** — Word Hunt and Grid Snap icons on platform game cards and each game hub.
- [x] **Difficulty setting** — dropdown picker persists across sessions; new puzzles respect Easy 4×4 / Medium 6×6 / Hard 8×8; stale in-progress saves no longer force a 4×4 resume.
