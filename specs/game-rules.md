# Game Rules

## Objective

Guess the secret **5-letter English word** within **6 attempts**.

## Guess rules

- Each guess is exactly **5 letters**.
- Each guess must appear in the **allowed guesses** word list.
- Proper nouns, abbreviations, and non-dictionary entries are rejected.
- The same word may be guessed more than once.

## Scoring (per guess)

After a valid guess is submitted, each tile receives one of three states:

| State | Color | Condition |
|-------|-------|-----------|
| Correct | Teal | Letter matches the secret word at that position |
| Present | Amber | Letter exists in the secret word but not at that position |
| Absent | Slate | Letter does not appear in the secret word (given remaining counts) |

### Duplicate letters

Scoring follows standard Wordle rules:

1. Mark **correct** (teal) positions first.
2. For remaining positions, mark **present** (amber) only while the secret word still has unmatched occurrences of that letter.
3. Any excess duplicate letters in the guess are marked **absent** (slate).

**Example:** Secret `APPLE`, guess `PAPER`

| Position | Letter | State |
|----------|--------|-------|
| 0 | P | Present |
| 1 | A | Present |
| 2 | P | Correct |
| 3 | E | Present |
| 4 | R | Absent |

**Example:** Secret `CRANE`, guess `REACT`

| Position | Letter | State |
|----------|--------|-------|
| 0 | R | Present |
| 1 | E | Present |
| 2 | A | Correct |
| 3 | C | Present |
| 4 | T | Absent |

## Win and loss

| Outcome | Condition |
|---------|-----------|
| Win | Guess matches the secret word before using all 6 rows |
| Loss | 6 guesses submitted without matching the secret word |

On loss, the secret word is revealed to the player.

## Secret word selection

### Practice mode

- Each new practice game picks a **random word** from the **answer word list**.
- Words are selected uniformly at random.

### Daily mode

- One shared word per **calendar day** (device local timezone).
- Derived deterministically from date via `getDailyWord(date)` in `src/games/word-hunt/core/dailyWord.ts`. A seeded hash spreads consecutive calendar days across the full answer list (not alphabetically).
- Resets at local midnight; in-progress daily games from a prior day do not resume.

### In-progress persistence

- Active daily and practice games (status `playing`) persist locally across navigation and app restarts.
- Completed games (won/lost) clear saved state.

## Word lists

Two bundled lists:

| List | Purpose | Approximate size |
|------|---------|------------------|
| Answer words | Secret words for new games | ~8,600 five-letter words |
| Allowed guesses | Validates player input | ~8,600 five-letter words |

**Allowed guesses** and **answer words** use the **ENABLE-1** word-list standard (same family as Wordle). Both files are generated by `npm run build:words` from `src/data/sources/enable1.txt` or the system dictionary. See [src/data/dictionary.md](../src/data/dictionary.md).

## Hard mode

When enabled (Settings), every subsequent guess must:

- Keep all **teal** letters in the same positions they were revealed.
- Include every **amber** letter somewhere in the guess (position may differ).

Violations reject the guess with a shake and message; the row does not advance.
